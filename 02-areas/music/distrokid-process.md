# Distrokid augšupielādes process

## Konts
- **Plāns:** Distrokid Ultimate (5 mākslinieki)
- **Email:** janis.jekabsons04@gmail.com

## KRITISKĀ secība (NEDRĪKST mainīt)

Apple Music sadaļa parādās TIKAI pēc audio faila augšupielādes!

```
1.  Artist/band name         ← teksts no profila
2.  Record label             ← teksts no profila
3.  Language                 ← dropdown no profila
4.  Primary genre            ← dropdown no profila
5.  Secondary genre          ← dropdown no profila
6.  AUGŠUPIELĀDĒ WAV failus  ← faili no Google Drive mapes
7.  AUGŠUPIELĀDĒ cover.jpg   ← fails no tās pašas mapes
8.  Song titles              ← no WAV failu nosaukumiem
9.  Songwriter info          ← no profila (katrai dziesmai)
10. Explicit: No             ← default (katrai dziesmai)
11. [Apple Music parādās]
12. Performer                ← no profila (katrai dziesmai)
13. Producer                 ← no profila (katrai dziesmai)
```

## Kas TIEK izlaists
- ❌ Release date — Jānis ievada pats
- ❌ Submit poga — NETIEK nospiesta automātiski

## Pauzes (anti-bot)
- Starp teksta laukiem: 2-3 sekundes
- Pēc failu augšupielādes: 5-8 sekundes
- Pēc dropdown izvēles: 1-2 sekundes

## Automatizācija
Chrome extension `suno-distrokid-extension` automatizē procesu.
Lietotājs tikai izvēlas albuma mapi un nospiež "Sākt aizpildīšanu".
