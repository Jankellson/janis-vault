## Kad pievienot vairāk

Tikai tad, kad konkrētā situācijā **uzreiz pamani**, ka derētu kāds no 19 pārējiem:
- Six Thinking Hats — grupu lēmumi (tu strādā viens, retāk vajag)
- Ishikawa — fizisks process kas nestrādā (varbūt VinkaFit klientu konversija)
- Decision Matrix — vairāki kandidāti pēc daudziem kritērijiem
- Minto Pyramid — gatavojot prezentāciju vai email
- Second-order thinking — pirms iet "all in" uz vienu virzienu

Tad pievienojam pa vienam, nevis visus uzreiz.
